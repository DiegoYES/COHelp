COHelp es una aplicación desarrollada en Android Studio que ayuda a los usuarios a registrar sus acciones ecológicas diarias
como usar transporte sostenible, reciclar o ahorrar electricidad, calcular el ahorro de co2 generado por estas acciones, 
y motivar su continuidad mediante desafíos interactivos y visualización de progreso.
Características
Registro de Hábitos Rápido: Tenemos una interfaz sencilla para ingresar la cantidad de acciones realizadas como km recorridos en bici y  kg reciclados.
Cálculo Automático de co2: Se usan factores de emisión predefinidos para calcular el impacto ambiental de cada acción registrada.
Dashboard Interactivo: Muestra el total de co2 ahorrado en el día y una gráfica semanal de barras para visualizar la tendencia de ahorro.
Desafíos Ambientales: Sección interactiva con retos ecológicos que el usuario puede marcar como completados, fomentando la adopción de nuevas prácticas.
Historial Detallado: Permite ver y eliminar registros pasados de hábitos, incluyendo la fecha, el tipo de hábito y el co2 ahorrado.
Notificaciones Diarias: Programa alarmas para recordar al usuario registrar sus hábitos.
